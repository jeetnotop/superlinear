# Example Documents for Docs REPL

This directory contains example documents for testing the `spl docs` REPL.

## Usage

```bash
spl docs wiki
/add examples/docs/wiki
```

## Contents

### wiki/

Eight Wikipedia articles about machine learning and neural networks:

- `artificial_intelligence.txt`
- `attention_machine_learning.txt`
- `byte_pair_encoding.txt`
- `large_language_model.txt`
- `long_short_term_memory.txt`
- `recurrent_neural_network.txt`
- `state_space_representation.txt`
- `transformer_deep_learning.txt`
